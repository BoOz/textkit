===========
tokens2stem
===========

Description
===========

::

    Usage: textkit tokens2stem [OPTIONS] [TOKENS]
    
      Stem a list of tokens to get their root.
    
    Options:
      -a, --algorithm [porter|lancaster|snowball|wordnet]
                                      Specify which stemming algorithm to use.
                                      [default: porter]
      --help                          Show this message and exit.
